  STEP 1 - Create GitHub account
    Go to https://github.com → Sign Up (free)

  STEP 2 - Upload code to GitHub
    a. Go to https://github.com/new
    b. Name: father-charles  → Create repository
    c. Click "uploading an existing file"
    d. Drag ALL files from this folder (except node_modules)
    e. Click "Commit changes"

  STEP 3 - Deploy on Railway
    a. Go to https://railway.app
    b. Click "Start a New Project"
    c. Sign in with GitHub
    d. Click "Deploy from GitHub repo"
    e. Select father-charles
    f. Wait for build to finish
    g. Click Settings → Networking → Generate Domain
    h. Your live URL appears! e.g. father-charles.up.railway.app
